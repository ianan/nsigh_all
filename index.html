Introduction

The sswidl code in this project creates an overview plot containing the time profile of the NuSTAR livetime (and optionally the CHU state), GOES and RHESSI fluxes. This data is stored within the project in *.dat files but the code can also generate these files from a local copy of the NuSTAR data and by downloading the GOES and RHESSI data.

Note that the features seen in the RHESSI and GOES lightcurves might not have been observed with NuSTAR as it does not have a full disk FoV.

Optionally the code can output a second image in which the GOES lightcurve is plotted with the NuSTAR observation times highlighted, e.g.
plot_ns_sun_lc, obsname='201411',gav=5,/goes,/gesnlog
