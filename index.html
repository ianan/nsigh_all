Introduction

This SSW code project creates overview plots of the NuSTAR solar observations, producing time profiles of the NuSTAR livetime (and optionally the CHU state), GOES and RHESSI fluxes. This data is stored within the project in *.dat files but the code can also generate them from the original data. Note that the features seen in the RHESSI and GOES lightcurves might not have been observed with NuSTAR as it does not have a full disk FoV.

It also produces overview maps and info files about each pointing (csv info files, avialable here). The maps and info files are created using only the grade 0 events, all the CHU combinations (with no pointing correction), >2.5 keV and each telescope module separately (FPMA and FPMB). Below are the combined maps per target as well as links to directories containing the maps for each pointing or orbit.
