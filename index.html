Introduction

The sswidl code in this project creates overview plots about the NuSTAR solar observations. It produces time profiles of the NuSTAR livetime (and optionally the CHU state), GOES and RHESSI fluxes. This data is stored within the project in *.dat files but the code can also generate these files from a local copy of the NuSTAR data and by downloading the GOES and RHESSI data. Original NOAA/SWPC GOES plots are avilable here or here. Note that the features seen in the RHESSI and GOES lightcurves might not have been observed with NuSTAR as it does not have a full disk FoV.

It also produces overview maps as well as info files about each pointing (csv info files, avialable here). The maps are created using only the grade 0 events, all the CHU combinations (with no pointing correction), >2.5 keV and each telescope module separately (FPMA and FPMB). The info files are created with these maps so the information they contain are from the same setup, i.e. times and pointing position are for all CHUs.
