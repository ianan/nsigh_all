Introduction

SSW code project creates overview plots/maps/info of the NuSTAR solar observations. The time profiles of the NuSTAR livetime (and optionally the CHU state), GOES and RHESSI fluxes, are in the project as *.dat files but the code can also generate them from the original data. Note that the features seen in the RHESSI and GOES lightcurves might not have been observed with NuSTAR as it does not have a full disk FoV. The maps and info (csv info files, avialable here) files are created using only the grade 0 events, all the CHU combinations (with no pointing correction), >2.5 keV and each telescope module separately (FPMA and FPMB). Below are the combined maps per target as well as links to directories containing the maps per pointing/orbit.
